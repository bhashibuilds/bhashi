Signify — Real-Time ASL Detection & Letter Recognition

Machine Learning, Computer Vision, Accessibility, AI

Case Study • 2025

Setting the Scene

Sign language is essential for Deaf and hard-of-hearing communities, but real-time recognition remains limited. Signify is a machine learning application that detects American Sign Language gestures and translates them into letters in real-time. The goal is to bridge communication gaps and provide developers with a complete ASL recognition pipeline.

Technologies Used

Signify was developed in Python and JavaScript (for web prototypes), leveraging TensorFlow.js and MediaPipe Hands for real-time hand tracking and gesture detection. OpenCV handles webcam integration and visualization, while HTML5 & CSS create the user interface.

Additional technologies and techniques include:

  • Landmark normalization
  • Data export in JSON for ML model training and sequence recording for dynamic gestures

Beta Version

I am in the process of implementing cloud storage for training data, allowing the model to incorporate contributions from external users. Collecting more diverse data is essential at this stage to improve the accuracy and reliability of ASL letter recognition.

Reflection & Next Steps

Building Signify allowed me to explore the full ASL recognition pipeline: from basic hand detection using MediaPipe to letter-level recognition with neural networks. Key takeaways include the importance of consistent landmark data, real-time performance optimization, and smoothing techniques to avoid flickering predictions.

Next steps involve more training, implementing word-level recognition, grammar-aware suggestions, and expanding support to multiple sign languages. Additionally, a user-friendly training data collection workflow can help build custom models for different users or environments.
